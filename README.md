# О проекте

<img width="1520" height="580" alt="image" src="https://github.com/user-attachments/assets/26d06923-13a3-4bbf-8f1f-66b4a039e082" />

FarmClicker (AnnaFarm) - это казуальная тайм-менеджмент игра, где игрок развивает собственную ферму, управляя циклами роста растений и эволюцией животных.
Игроку предстоит оптимизировать производство ресурсов опираясь на время и полученные задачи.
